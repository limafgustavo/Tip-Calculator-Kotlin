# TipCalculator

TipCalculator is a simple Android app that calculates the tip amount based on the total bill and the selected tip percentage. It allows users to adjust the tip percentage using a seek bar and displays the calculated tip amount and the total bill with the tip included in real-time.

## Used Technologies

- View Binding: Simplifies interaction with views in the layout and provides type-safe access to UI elements.
- TextWatcher Interface: Monitors changes to the total bill amount entered by the user and triggers the tip calculation accordingly.
- SeekBar.OnSeekBarChangeListener Interface: Tracks changes to the selected tip percentage using the seek bar.

## Features

- Calculate the tip based on the total bill and selected tip percentage.
- Easily adjust the tip percentage using a user-friendly seek bar.
- See real-time updates of the tip amount and the total bill with the tip included.

## How to Use

1. Enter the total bill amount in the provided field.
2. Use the seek bar to adjust the desired tip percentage.
3. The app will instantly display the calculated tip amount and the total bill with the tip included.

## Build and Run

1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Build and run the app on an Android emulator or a physical device.
